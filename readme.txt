Tags: car, mobile.de, random, merchant
Stable tag: 1.0.1

This plugin provides a predetermined number of advertisements from the merchant site Mobile.de. You can set the position of the output and a random selection for the advertisements is available.

== Installation ==
1. Upload the directory `wpRandomCar´ with included files to the `/wp-content/plugins/´ directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Edit the config and use [randomcar] within your posts and sites for displaying.
